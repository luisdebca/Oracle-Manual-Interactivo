Proceso de configuracion de la base de datos de organizacion Proyesa SA de CV .

Procesos para la configuracion del la base de datos Proyesa

1. Este paso es opcional ya que estaremos ingresando CMD para hacer cambios en la contraseña por si al realizar la instalacion se nos olvidamos de nuestra contraseña de super usuario. Estando en CMD ingresaremos el comando: sqlplus "/as sysdba" y posterior digitaremos: Alter user System identified by 12345

2. Abriremos SQL Developer y como no tenemos ninguna conexion, realicemos una para el usuario administrador para que mendiante a ese podamos crear usuarios entre otros permisos que este tiene. Le darmos clic a boton con el simbolo "+" que se encuentra en el lado superior izquierdo para realizar la connexion.

3. Completamos la informacion requerida para realizar la conexion del usuario Administrador "System", le damos clic en Probar y si todo esta bien, nos indicara que todo esta correcto.

4. Teniendo creada la conexion del usuario administrador, accedemos a ella para que podamos realizar un usuario que le llamaremos PIBD, el cual al correr las sentencia nos indica que ha sido creado

5. Ya teniendo creado el usuario procederemos a realizar la bases de datos en donde se cargara el Script. Procedemos a seguir los mismos pasos que el punto 3. Cabe mencionar que este colocaremos el usuario que sera dedidacado solo a nuestro proyecto.

6. Procederemos a copia nuestro codigo Script y correrlo. Notaremos al momento el que SQL developer corre la instruccion va realizando los cambios
