Requerimientos para la instalación

Requerimientos de hardware

Requerimientos mínimos de hardware

Requerimientos Valor
Arquitectura del sistema Procesadores: AMD64 o Intel EM64T (Extended Memory 64 Technology)
Memoria Física 2 GB
Memoria Virtual (SWAP)
  • Si la memoria física está entre 2 GB y 16 GB, configure la memoria virtual en 1 veces el tamaño de la RAM
  • Si la memoria física tiene más de 16 GB, configure la memoria virtual en 16 GB
Espacio en disco 10 GB o más
Adaptador de video y Resolución de pantalla 256 colores, 1024 X 768

Requerimientos recomendados de hardware

Requerimientos Valor
Arquitectura del sistema Procesadores: AMD64 o Intel EM64T (Extended Memory 64 Technology)
Memoria Física 8 GB
Espacio en disco 12 GB o más
Adaptador de video y Resolución de pantalla 256 colores, 1280 X 720

Java JDK o JRE 1.8 o mayor
Descompresor de archivos Para extraer los archivos de instalación .zip

Es importante saber que Oracle Database 18c solo está disponible para sistemas Windows de 64 bits, al igual que Oracle 12c. Por lo tanto, no podemos instalar Oracle Database 18c en nuestra PC si está ejecutando un sistema operativo Windows de 32 bits.
